## 🆘 Common Issues

| Issue | Solution |
|-------|----------|
| Port in use | Change port in vite.config.js |
| Cannot find module | Run `npm install` |
| Dark mode not working | Check localStorage permissions |
| Drag & drop fails | Ensure dnd-kit is installed |
| API connection error | Check VITE_API_URL in .env |
